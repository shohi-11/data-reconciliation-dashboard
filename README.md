#  Data Reconciliation Dashboard

A web-based dashboard designed to upload CSV files, map columns dynamically, and visualize reconciliation results such as matched, unmatched, and duplicate records through interactive charts and summaries.


##  Features

-  Upload CSV files for reconciliation
-  Dynamic column mapping (Transaction ID, Amount, Reference, Date)
-  Validation of mapped columns
-  Data preview before processing
-  Visual dashboards with:
  - Total Records
  - Matched Records
  - Unmatched Records
  - Duplicate Records
-  Bar chart & donut chart for status distribution
-  Clean, responsive UI



##  Tech Stack

**Frontend**
- HTML5
- CSS3
- JavaScript
- Bootstrap
- Chart.js
